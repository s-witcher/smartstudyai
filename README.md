# 🧠 SmartStudyAI

SmartStudyAI helps students learn faster by:
- Summarizing research papers
- Generating quizzes
- Creating flashcards

## 🚀 Features
- Ingests PDFs or text files
- Uses OpenAI GPT models
- Modular design (ingestion, provider, algorithms)
